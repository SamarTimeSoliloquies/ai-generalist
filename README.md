# ai-generalist
AI-powered RAG consultant for Law, Accounting &amp; Medicine — hallucinaton-free document research and agentic drafting using Next.js, FastAPI, LangGraph, pgvector, and Supabase

AI Generalist
An AI-powered web consultant and document drafting assistant for Law, Accounting, and Medicine. Built to eliminate AI hallucinations using Retrieval-Augmented Generation (RAG) and Agentic Loops.

The Problem
Professionals in law, accounting, and medicine lose hours searching through massive documents and drafting complex paperwork. Standard AI cannot be trusted because it hallucinates — it fabricates facts confidently.

The Solution
AI Generalist restricts the AI to only answer based on real, official documents uploaded into the system. It then uses Agentic Loops to automatically draft, review, and format professional documents like legal briefs, tax memos, and patient summaries.

Tech Stack
Frontend: Next.js (React)
Backend: FastAPI (Python)
AI Orchestration: LangGraph + LangChain
Relational Database: Supabase (PostgreSQL)
Vector Database: pgvector
Cache: Redis
File Storage: AWS S3 / Supabase Storage
